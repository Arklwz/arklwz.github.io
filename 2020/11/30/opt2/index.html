菲涅尔衍射

菲涅尔波带法

  • 以$P_0$点为中心，半径分别为$z_1,z_1+\frac{\lambda}{2},z_1+\lambda,…z_1+\frac{j\lambda}{2}$作球面，与$\Sigma$相交成圆，将$\Sigma$划分为各个环带
    其中每个环带对$P_0$点产生的振幅可以表示为
    $|\tilde{E_j}| = C\frac{A_j}{r_j}\frac{1+cos\theta}{2}$
    由于$A_j$为两个相邻圆面积之差，大约为常数 $\pi z_1\lambda$
  • 各波带在$ P_0 $点产生的复振幅总和
    $ \tilde{E} = |\tilde{E_1}| - |\tilde{E_2}| + |\tilde{E_3}|+…-(-1)^n|\tilde{E_n}|$
    近似有$|\tilde{E_2}| = |\tilde{E_1}|/2 + |\tilde{E_3}|/2$, $|\tilde{E_4}| = |\tilde{E_3}|/2 + |\tilde{E_5}|/2$,…
    得到
    $\tilde{E} = \frac{|\tilde{E_1}|}{2} \pm \frac{|\tilde{E_n}|}{2} $ (n取奇数为+，取偶数为-)
    由此可见，当n为奇数时$P_0$点强度较大，n为偶数时$P_0$点强度较小，逐渐开大或缩小圆孔，在$P_0$点将可以看到明暗交替的变化
  • 当圆孔非常大时候，或者不存在圆孔衍射屏时，$ |\tilde{E_n}| = 0 $
    则$ \tilde{E_\infty} = \frac{\tilde{E_1}}{2} $
    可见此时$P_0$点复振幅为第一波带产生复振幅的一半

菲涅尔波带片(菲涅尔透镜)

  • 特殊光阑，使得奇数波带畅通无阻，偶数波带完全被阻挡
  • 例如，包含20个波带的光阑，让奇数波带通过
    $\tilde{E} = |\tilde{E_1}| + |\tilde{E_3}|+…+|\tilde{E_19}| \approx 10|\tilde{E_1}| = 20|\tilde{E_\infty}|$
    结果使光强相对于无光阑时扩大400倍
  • 第J个波带外圈半径 $\rho = \sqrt{jz_1\lambda}$, 因此波带片焦距$ f = z_1 = \frac{\rho ^2_j}{j\lambda}$
  • 波带片除了有主焦点$P_0$,还有一系列光强较小的焦点$P_1$，$P_2$,…他们分别距离波带片$f/3, f/5,…$,此外还有很多与实焦点对称的虚焦点

典型孔径夫琅和费衍射

夫琅和费衍射公式的意义

  • 夫琅和费衍射公式可以表示为
    $\tilde{E}(P) = c\iint_\Sigma \tilde{E}(x_1,y_1)exp[-ik(lx_1+\omega y_1)]dx_1 dy_1$
    $ = c\iint_\Sigma \tilde{E}(x_1,y_1)exp[-i\frac{2\pi}{\lambda f}(xx_1+yy_1)]dx_1 dy_1$
  • 公式表明
    (1) P点的复振幅取决于孔径面上所有Q方向衍射光在该点的光程差
    (2)$\tilde{E}(P) = c\iint_\Sigma \tilde{E}(x_1,y_1)exp[-i2\pi(ux_1+vy_1)]dx_1 dy_1$
     焦面场上的衍射分布就是孔径面上复振幅分布的一个傅立叶变换
     傅立叶变换模拟运算可以用衍射变换实现
     衍射问题可用傅立叶变换方法处理

矩孔衍射

  • 强度分布
    $ I = |\tilde{E}|^2 = I_0(\frac{sin\frac{kla}{2}}{\frac{kla}{2}})^2(\frac{sin\frac{k\omega b}{2}}{\frac{k\omega b}{2}})^2 = I_0(\frac{sin\alpha}{\alpha})^2(\frac{sin\beta}{\beta})^2$
    其中$\alpha = \frac{kla}{2} = \frac{\pi}{\lambda}asin\theta_x, \beta = \frac{k\omega b}{2} = \frac{\pi}{\lambda}bsin\theta_y$
  • 衍射图案(分别由x,y轴进行分析)
    (1)极小位置
    当 $ \alpha = n\pi $时
    (2)极大位置
    当 $ \alpha = 0 $主极大
    当 $ tan\alpha = \alpha $次级大
    (3)衍射效应正比入射波长，反比于孔径线度。
    由于$ sin\theta_x = \frac{n\lambda}{a}$ 条纹角宽度$\Delta\theta_x = \frac{\lambda}{a}$
    矩孔的夫琅和费衍射图样：中央零级在点源的几何像位置，中央亮斑集中了全部能量的80%。周围是一些x、y方向上等间距的暗线，相邻暗线之间距反比于矩孔线度。

圆孔的夫琅和费衍射

  • 强度分布
    $ I(P) = I_0[\frac{2J_1(Z}{Z}]^2, Z = ka\theta$
  • 衍射图样
    (1)圆孔衍射图样是中心明亮，明暗相间的同心圆环，在θ=0（即点源几何像）处强度最大，随θ变化，会出现强度的极大、极小
    (2)极大、极小值分布不等间距
    第一极小位置$ Z = 1.22\pi$, 即$\theta = \frac{0.61\lambda}{a} = \frac{1.22\lambda}{D}$ 为艾里斑半径
    次级大$ J_2(Z) = 0 $
    (3)衍射效应与孔径线度成反比，与波长成正比

光学成像系统的衍射和分辨本领

成像系统的衍射现象

在像面观察的夫琅和费衍射

  • 成像系统对近处点物在像面所成的夫琅和费衍射像等价于平面波入射时，在$ f = R$的透镜焦面上产生的孔径夫琅和费衍射分布，这提供一种用会聚光照明在像面得到孔径频谱的方法。

光学成像系统的分辨率

  • 瑞利判据：当一个点物衍射图样的中央极大刚好为另一个点物衍射图样的第一极小时，认为此时两点物可分辨。
  • 典型光学系统的分辨本领
    (1)人眼的最小可分辨角$ \alpha = \frac{1.22\lambda}{d} $
    (2)望远物镜最小可分辨角$ \alpha = \theta = \frac{1.22\lambda}{D} $
    (3)照相物镜分辨本领$ N = \frac{D}{1.22\lambda f} $(底片上每毫米可分辨的线条数)
    (4)显微物镜的最小可分辨距$ \varepsilon = \frac{0.61\lambda}{n\sin u}$

多缝的夫琅和费衍射

多缝的强度分布公式

  • $ I(P) = I_0(\frac{sin\alpha}{\alpha})^2(\frac{sin\frac{N\delta}{2}}{sin\frac{\delta}{2}})^2$
    其中$\alpha = \frac{kla}{2}$ 或 $ asin\theta = n\lambda$
      $\delta = \frac{2\pi}{\lambda}dsin\theta$ 或 $dsin\theta = m\lambda $
  • 该式包含单缝单缝衍射和多缝干涉

多缝衍射图样的特征

  • 由$(\frac{sin\frac{N\delta}{2}}{sin\frac{\delta}{2}})^2$取干涉极大极小值位置
    (1)当$\delta = 2m\pi$ 或 $ dsin\theta = m\lambda $ 时出现干涉主极大，且强度为单缝时的$N^2$倍
    (2)当$ \delta = \frac{2m\pi}{N}$ 或 $ dsin\theta = (m + \frac{m’}{N})\lambda$时出现干涉极小值
    (3)相邻两个极小值间有一次极大，因而相邻两主极大间有(N－1)个极小值，(N－2)个次极大.
  • $(\frac{sin\alpha}{\alpha})^2$的调制作用
    (1)当$\alpha = n\pi$ 或 $asin\theta = n\lambda$ 时取得衍射极小
  • 缝数N对衍射分布的影响
    (1)N增加，使各相邻主极大数目增加
    (2)N增加，使各主极大的强度增强N²倍
    (3)N增加，各级主极大（条纹）变细变亮
    主极大角半宽$ \Delta\theta = \frac{\lambda}{Ndcos\theta_m} $

衍射光栅

光栅的分光性能

  • 光栅方程：$d(sini\pm sin\theta) = m\lambda$(与法线同侧取正，异侧取负)
  • 光栅的色散：不同波长的同级主极大对应不同的衍射角的现象
    (1)角色散：表示波长相差0.1nm的两条谱线分开的角距离
    $\frac{d\theta}{d\lambda} = \frac{m}{dcos\theta}$
    (2)线色散：波长相差0.1nm的两谱线在焦面上分开的线距离
    $\frac{dl}{d\lambda} = f\frac{m}{dcos\theta}$
    (3)匀排光谱：色散是均匀的光栅光谱（θ角不大时，视Cosθ不随θ而变）
  • 光栅的分辨本领
    $A = \frac{\lambda}{\Delta\lambda} = mN$
  • 光栅的自由光谱区
    $\Delta\lambda = \frac{\lambda}{m}$
