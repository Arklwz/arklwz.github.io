应用光学——典型光学系统

发表于 2020-11-30 分类于 学习所悟 阅读次数：

  几何光学典型光学系统，主要包括眼镜、放大镜、显微镜、望远镜、摄影系统和投影系统几部分组成。

以后有机会把典型光学系统的图贴上，暂时算更新完了
  该部分《841工程光学基础》考纲内容如下

熟练掌握各种典型光学系统的成像原理、光束限制、放大倍率、分辨本领，掌握显微镜、投影系统及其照明系统、望远镜和转像系统的关系，能够解决典型光学系统的外形尺寸计算问题。

眼睛

眼睛的结构——成像光学系统

眼睛的调节及矫正

  • 眼睛的调节能力用能清晰调焦的极限距离表示，远点距离$l_r$和近点距离$l_p$，其倒数$1/l_r=R$,$l/l_p=P$分别表示远点和近点的发散度或汇聚度，其单位为屈光度(D)。
  • 眼睛的调节能力以远点和近点的倒数差来衡量，即$1/l_r-1/l_p = R - P = \bar{A}$
  • 正常眼的远点在无限远，眼睛光学系统的后焦点在视网膜上。
  • 近视眼远点位于眼前有限距离，远视眼远点位于眼后有限距离
  • 散光主要是由像散引起的

眼睛——辐射接收器

眼睛的分辨率

  • 视网膜上最小鉴别距离等于两神经细胞直径，不小于$0.006mm$

  • 物体对人眼的张角为视角，人眼能分辨的物体间最小视角称作视角鉴别率$\varepsilon$

    $tan\varepsilon = \frac{0.006}{f’}\times206265’’$

  • 视角敏锐度 = $1/\varepsilon$

眼睛的对准精度

眼睛的景深

双目立体视觉

  • 当双目观察物点A时，两眼的视轴对准A点，两视轴之间的夹角$\theta$称为视差角
    $\theta_A = b/L$
    其中b为视觉基线一般为62mm，L物点A到基线的距离
  • 不同物体对应不同的视差角，其差异$\Delta\theta$被称为立体视差，简称视差
  • 双眼能分辨两点间的最短深度被称为立体视觉阈$\Delta L$
    $\Delta L = \Delta\theta L^2/b$

放大镜

视觉放大率

  • 为用仪器观察物体在视网膜的像高比用人眼直接观察物体在视网膜上的像高
    $\Gamma = y’_i/y’_e = \frac{tan\omega’}{tan\omega}$
  • 当眼睛调焦无穷远时 $\Gamma _0= \frac{250}{f’}$
  • 当眼睛调焦在明视距离时 $\Gamma = \frac{250}{f’}+1$

光束限制和线视场

  • 眼瞳：孔阑、出瞳
  • 放大镜框： 视场光阑、渐晕光阑
  • 当渐晕50%时，线视场$2y = \frac{500h}{\Gamma _0 P’}$
    由此可知，放大镜的倍率越小，镜框越大，P’(距离)越近，线视场越大

显微镜系统

显微镜的视觉放大率

$\Gamma = \frac{tan\omega’}{tan\omega} = -\frac{(250mm)\Delta}{f’_o f’_e} = \beta\Gamma_e$
把显微镜看做一个组合透镜系统 其组合焦距 $f’ = -\frac{f’_o f’_e}{Delta}$ 那么$\Gamma _0= \frac{250mm}{f’}$
这说明显微镜实质上与放大镜相同，显微镜可以看作组合的放大镜

显微镜的线视场

显微镜的线视场取决于放在目镜前焦平面上的视场光阑大小D,物体经物镜成像于视阑（目镜前焦面）$ 2y = D/\beta$
而视阑的大小又要和目镜的视场角一致$D = 2f’_etan\omega’$
联立得到线视场表达式$2y = \frac{500tan\omega’}{\Gamma}$
可见，在选定目镜后(视场角给定)，显微镜的视觉放大率越大，线视场越小

显微镜的出瞳直径

$D’ = \frac{500NA}{\Gamma}$,其中$NA = nsinu$ 为物镜的数值孔径
显微镜的出瞳直径很小，一般小于眼瞳，只有在低倍时才能达到眼瞳直径

显微镜的分辨率和有效放大率

  • 瑞利判断，相邻两个点之间的距离等于艾里斑半径时恰好被光学系统分辨，$ a = 0.61\lambda/n’sinu’$
  • 道威判断，相邻两个点之间的两衍射中心距为0.85a时恰好被光学系统分辨
  • 瑞利分辨率标准比较保守，通常用道威分辨率值作为光学系统的目视衍射分辨率
    $\sigma = 0.85a/\beta = 0.5\lambda/nsinu$
  • 有效放大率，对于555nm平均波长的入射光，$500NA < \Gamma < 1000NA$

显微镜的景深

$ 2\Delta = \frac{250n\varepsilon}{\Gamma NA} $
数值孔径越大，放大倍率越高，景深越小

显微镜的照明方法

  • 临界照明（一个聚光镜）（窗对窗，瞳对瞳）
    光源->物面
    聚光镜前焦面上的可变光阑->显微物镜后焦面上的孔径光阑
    照明系统应满足：聚光镜的像方孔径角大于等于显微物镜的物方孔径角
    缺点：不均匀照明
  • 柯拉照明（两个聚光镜）(窗对瞳，瞳对窗)
    光源->聚光物镜物方焦点->平行光->孔阑
    前置物镜（光阑）->物面
    柯拉照明是一种均匀照明
  • 如果要观察小于分辨极限的微小质点，应采用暗视场照明。

显微镜的物镜

  • 主要参数:焦距、孔径、视场角，分别决定放大率、数值孔径、线视场
  • 由于显微镜要求分辨率高，则数值孔径大，放大率要与之相适应，因而物镜的放大率也要相应匹配，并在规定机械筒长下使用。
  • 随着放大倍数增高，结构变复杂。低倍物镜可以用双胶合，中倍物镜用双双胶合，高倍物镜用双胶合＋前片，数值孔径更大的阿贝物镜则需要浸油。

望远镜系统

  • 望远镜视觉放大率$\Gamma = \frac{tan\omega’}{tan\omega} = \gamma$
    $\Gamma = -f’_o/f’_e = -D/D’$
  • 垂轴放大率：$\Gamma = 1/\beta$

望远系统的分辨率及工作放大率

  • 极限分辨角：$ \varphi = \frac{a}{f’_o} = \frac{0.61\lambda }{n’sinu’f’_o} $
    取 $ n = 1, sinu’ = D/2f’_o, \lambda = 555nm $
    得到$ \varphi = 140’’/D $
  • 要求$ \Gamma\varphi>=60’’-70’’$ 所以$\Gamma >= D/2 $为正常放大率,此时出瞳为2mm与眼瞳相当
  • 瞄准精度

望远镜的主观亮度

  • 点光源：高倍望远镜具有增大点光源主观亮度的作用。当望远镜入瞳一定时，随倍数增大出瞳逐渐减小，至出瞳与眼瞳相当时，继续增大放大倍数不再影响主观亮度。
  • 拓展光源：眼睛通过望远镜观察扩展光源的主观亮度取决于网膜上的照度。
  • 星星作为点光源，主观亮度增大；天空是扩展光源，主观亮度减小。当望远镜倍数足够高时，可能在白天看见星星。

望远镜的光束限制

  • 伽利略望远镜（正+负）
    眼瞳为孔径光阑，物镜为渐晕光阑
  • 开普勒望远镜（负+负）
    物镜为孔阑，目镜如拦光则为渐晕光阑

望远镜的物镜

望远镜的物镜属于长焦距中等孔径小视场系统，主要校正轴上点像差，有折射式、反射式、折反射式三种。折射式有双胶合物镜、双分离物镜、三片式物镜、内调焦物镜等。当物镜直径大于60mm时不宜胶合。反射式可以做到很大的入瞳直径，对材料无严格要求，筒长较短，完全无色差，但对表面质量要求更高，通常用非球面。折反射式以球面或非球面反射镜为基础，再加上用于校正像差的折射元件，可以做到很高的成像质量。

望远镜的目镜

  • 属于短焦距中等孔径大视场系统
  • 目镜应能作视度调节(公式待补充)

望远镜的转向系统

  • 棱镜转像系统：要求偶次反射，有利于减小筒长。
  • 透镜转像系统：适用于筒长较长时
    单组透镜转像系统，对于－1倍单组转像透镜，筒长加长了4倍转像透镜的焦距。其中场镜使物镜和转向透镜共轭
    双组透镜转向系统，如果两个转像透镜焦距相等，距离为d，则筒长增加了$d+2f’_a$ 转像系统的孔径光阑设在其内部，对称结构使垂轴像差自动为零。场镜使物镜与孔阑共轭

摄影系统

摄影物镜的光学特性

摄影物镜的光学特性主要由焦距$f’$,相对孔径$D/f’$,视场角$2\omega$。三者分别决定成像大小、像面照度、成像范围

  • 像的大小
    远处时$y’ = -f’tan\omega$,近处时$y’ = y\beta = yf’/x$
    焦距大视场小：特写镜头，远摄镜头
    焦距小视场大：全景镜头，广角镜头
  • 像面照度与相对孔径二次方成正比
    相对孔径大、强光镜头，相对孔径中等、普通镜头，相对孔径小、弱光镜头，
  • 视场角
    能摄入接收面的视场角，由接收面大小决定，$2y’$是接收面的对角线长

摄影系统中的光束限制

  • 孔阑：是特设的可变光阑，一般在镜头内部对称面附近。 光圈数（F#）
  • 视阑：接收面，如底片框或图像传感器成像面
  • 渐晕光阑：由于摄影光学系统视场、孔径都很大，远离孔径光阑的透镜常要拦光，否则易造成透镜口径太大，笨重不便，而且轴外宽光束像差也会影响像质。同样光圈数时不同视场渐晕不同，同样视场不同光圈数渐晕也不同。

摄影光学系统的景深和几何焦深

  • 摄影光学系统的景深：焦距越长，入瞳直径越大，景深越小。拍摄距离越大，景深越大
  • 几何焦深：F数越大，越允许调焦不准。几何焦深可作为轴向像差的允差。

摄影物镜的分辨率

  • 最小分辨距：$\sigma = \frac{1.22\lambda}{D/f’}$
  • 摄影系统的分辨率常用每毫米能区分的线条数表示, $N = \frac{1}{\sigma }$
  • 对于555nm波长，$N = 1475/F$

常见摄影物镜

  • 标准镜头的视场角和人眼相当，可以拍出人眼看起来很自然的照片，而不会在照片上附加任何感觉。
  • 广角镜头具有更大的视场角，善于营造方向线，拍出的照片具有空间延伸感。由于近大远小的透视效果，用于拍摄人物要注意人物的变形。
  • 鱼眼镜头是广角镜头的极端，它不满足理想的物像关系，而是具有很大的负畸变，可以把180度甚至更大的视场摄入画面。
  • 远摄镜头具有把距离拉近的效果，并且由于景深小，可以突出画面上的被摄主体，而使背景变得简洁。
  • 焦距在70mm至100mm左右的镜头在远摄镜头家族中焦距算是短的，特别适合拍摄人像特写，又叫人像镜头。
  • 焦距在100mm至135mm的中等焦距远摄镜头在微距下特别善于表现物体的细节与质感。
  • 焦距在200mm左右的中长焦距远摄镜头常用于拍摄远处的新闻画面，如报道体育比赛等。
  • 焦距为300至500mm甚至更长的远摄镜头是新闻报道、野生动物摄影的重要创作工具。
  • 一些镜头被称为快速镜头，它在同样焦距下具有更大的相对孔径，因而可以以更短的曝光时间捕捉画面。

投影系统

投影系统组成

  • 投影及放映光学系统由于要成放大像，为了保证一定的像面照度，通常要加照明系统。因此，其光学系统包括照明系统和成像系统两部分。
  • 照明系统的类型有透射光照明和反射光照明，前者包括电影放映机、放大机及一些透射照明投影系统，后者有反射投影仪、某些液晶投影系统等。

投影或放映系统和照明系统的关系

  • 如果要实现像面照度均匀，应采用类似于显微镜系统的柯拉照明方式，即照明系统的孔径光阑与成像系统的视场光阑共轭，照明系统的视场光阑与成像系统的孔径光阑共轭。
  • 在投影系统中，应该是光源像位于成像系统的孔阑处，而图片位于照明系统的孔阑处。
  • 如果把聚光镜和成像物镜都看成是单薄透镜的话，应该图片和聚光镜靠在一起，光源像与成像物镜重合。

像面照度

  • 正比于相对孔径的平方，反比于垂轴放大率的平方
