平面波的复振幅分布和空间频率

光场中任一平面上的复振幅分布

  • 空间复振幅分布 $ \tilde{E}(x,y,z) = Aexp[ik(xcos\alpha + ycos\beta + zcos\gamma)] $
  • 令$z = z_0$ 则可以得到 $ \tilde{E}(x,y) = A’exp[i\frac{2\pi}{\lambda}(xcos\alpha + ycos\beta)] $

复振幅分布的空间频率及物理意义

  • 空间频率 $ u = \frac{cos\alpha}{\lambda} = \frac{sin\theta_x}{\lambda}, v = \frac{cos\beta}{\lambda} = \frac{sin\theta_y}{\lambda}$
    则 $ \tilde{E}(x,y) = A’exp[i2\pi(ux + vy)] $
  • 空间频率的意义
    (1)空间频率表示复振幅分布的周期性
    (2)
    (3)光波场某一平面上的复振幅分布可以分解成许多不同基本周期分布，则表明包含着许多不同的（u,v）成分，对应着许多不同方向$(cos\alpha,cos\beta)$传播的平面波
    (4)（u,v）也可以表示不同强度分布的空间周期性

复杂复振幅按(u,v)的分解

  • $\tilde{E}(x,y) = \iint\tilde{E}(u,v)exp[i2\pi(ux+vy))]dudv, \tilde{E}(x,y) = F^-1{\tilde{E}(u,v)}$
  • $\tilde{E}(u,v) = \iint\tilde{E}(x,y)exp[-i2\pi(ux+vy))]dudv,
    \tilde{E}(u,v) = F{\tilde{E}(x,y)}$

衍射现象的傅里叶分析方法

夫琅和费近似下衍射场与孔径场的傅立叶变换关系

  • $\tilde{E}(x,y) = \tilde{E}(u,v) = F{\tilde{E}(x,y)}, u = \frac{x}{\lambda z_1} v = \frac{y}{\lambda z_1}$
    可以表示
    (1)衍射场的分布只须对孔径面复振幅分布做一个傅氏变换求取
    (2)焦面场（衍射场）上是不同方向传播的平面波的一个频谱分布，焦面即为频谱面

夫琅和费衍射举例

  • 矩孔和单缝衍射
    $\tilde{E}(x_1,y_1) = \tilde{E_0}(x_1,y_1)\tilde{t}(x_1,y_1) = rect(\frac{x_1}{a}) rect(\frac{y_1}{b}) $
    $\tilde{E}(u,v) = F{\tilde{E}(x_1,y_1)} = abSinc(au)Sinc(bv)$
    $I(x,y) = |\tilde{E}(u,v)|^2 = I_0 Sinc^2(\frac{ax}{\lambda z_1})Sinc(\frac{by}{\lambda z_1})$
    傅里叶变换相似定理
    $F{\tilde{E}(mx,ny)} = \frac{1}{mn}\tilde{E}(\frac{u}{m},\frac{v}{n})$
  • 双缝衍射
    傅里叶变换的相移定理
    $F{\tilde{E}(x-a,y-b)} = \tilde{E}(u,v)exp[-i2\pi(ua+vb)]$
    $\tilde{E}(x_1) = rect(\frac{x_1-\frac{d}{2}}{a}) + rect(\frac{x_1+\frac{d}{2}}{a})$
    $\tilde{E}(x) = F{\tilde{E}(x_1)} = F{rect(\frac{x_1}{a})}exp[-i2\pi u(-\frac{d}{2})] + F{rect(\frac{x_1}{a})}exp[-i2\pi u(\frac{d}{2})] $
    $ = 2acos(\pi ud)Sinc(au) = 2aSinc(\frac{ax}{\lambda z_1})cos(\frac{\pi ud}{\lambda z_1}) $
  • 正弦光栅
    无限长度时$ \tilde{t}(x_1) = 1 + cos2\pi u_0 x_1$
    $\tilde{E}(u) = F{\tilde{E}(x_1} = \delta(u) + \frac{1}{2}\delta(u-u_0) + \frac{1}{2}\delta(u+u_0)$
    有限长度时$ \tilde{t}(x_1) = (1 + cos2\pi u_0 x_1)rect(\frac{x_1}{L})$
    $\tilde{E}(u) = [\delta(u) + \frac{1}{2}\delta(u-u_0) + \frac{1}{2}\delta(u+u_0)]*Lsinc(Lu)$
    $ = Lsinc(Lu) + \frac{L}{2}sinc[L(u-u_0)] + \frac{L}{2}sinc[L(u+u_0)]$

透镜的傅里叶变换性质和成像性质

透镜的透射系数

  • $\tilde{t}(x_1,y_1) = exp[-ik\frac{x^2_1+y^2_1}{2f}]$
  • 注意
    (1)透镜为有限大小时，引入光瞳函数
    (2)对任一形式的入射光波，透镜给出相同形式的相位影响
    (3)具有与透镜相同形式的透明片，其作用类似于相同焦距的透镜
    (4)$f>0$为正透镜$f<0$为负透镜

透镜的傅里叶变换性质

  • 衍射屏紧靠透镜前表面
    后焦面上的光场分布是衍射屏平面复振幅分布的一个傅里叶变换，后焦面上某点振幅和相位取决于透过屏某一(u,v)分量的振幅和相位。
  • 衍射屏置于透镜前一定距离
    透镜后焦面上得到的也是衍射屏平面复振幅分布的傅里叶变换
  • 结论
    (1)单色平面波正入射时，不论衍射屏置于透镜何处，在透镜后焦面上将得到其频谱
    (2)当球面波照明时，物置于任何位置，透镜均起到傅里叶变换作用，但其频谱面在点源像面位置上。
    (3)存在孔径大小影响：孔径的衍射效应使像质模糊，透镜是低通滤波器，存在渐晕现象

相干成像系统分析和相干传递函数

点物成像和线性不变系统

  • 点扩散函数： 单位复振幅或光强的点物经过系统所产生的像斑的复振幅或光强的分布
    表示为：$ h(x,y;x’,y’)$
  • 光学系统的线性和空间不变性
    (1)光学系统的线性性：
    线性系统：其总输出为各单个输出的线性组合的系统。
    光学系统的线性性：一个复杂的光场分布可以分解成许多简单基元分布的线性叠加。
    相干光学成像系统：对复振幅分布呈线性。
    非相干光学成像系统：对强度分布呈线性。
    (2)空间不变性：
    成像系统的空间不变性：点扩散函数的形式不随场面上物点位置而变的系统。点扩散函数只取决于观察点与几何像点相对位置。
    此时$h(x,y;x’,y’) = h(x’-x,y’-y)$

扩展物体的成像

  • 对于线性空不变系统，扩展物体的光场分布可以看成许多点物经系统形成的点扩散函数的线性组合。
  • $ g(x’)=\int O(x)h(x’-x)dx = O(x’)*h(x’)$
  • 物的像的光场分布是物的光场与系统点扩散函数的卷积

相干传递函数（CTF）

CTF的意义

二维物体的像g(x’,y’) = O(x’,y’)*h(x’,y’)
$ Gc(u,v) = F{g(x’,y’)},Oc(u,v) = F{o(x’,y’)},Hc(u,v) = F{h(x’,y’)}$
根据卷积定理有
$Hc(u,v) = \frac{Gc(u,v)}{Oc(u,v)}$
结论：物函数->物的频谱×相干传递函数->像的频谱->像函数

CTF的求取

  • 光瞳函数-> 点扩散函数->相干传递函数
    $h(x’,y’) = Aexp[ik\frac{x’^2+y’^2}{2l’}]\iint P(\xi,\eta)exp[-2\pi(u’\xi+v’\eta)]d\xi d\eta, u’ = \frac{x’}{\lambda l’},v’ = \frac{y’}{\lambda l’}$
    故$h(x’,y’) = F{P(\lambda l’u,\lambda l’v)}$
    所以$Hc(u,v) = FF{P(\lambda l’u,\lambda l’v)} = {P(-\lambda l’u,-\lambda l’v)}$
    考虑到光瞳函数具有对称性
    $Hc(u,v) = P(\lambda l’u,\lambda l’v)$
  • 讨论
    (1)$Hc(u,v)$ 为01函数，存在一个截止频率$\rho_M = \sqrt{u^2+v^2}$
    (2)光学系统是低通滤波器
  • 例：
    $P(\xi,\eta = circ(\frac{\sqrt{\xi^2+\eta^2}}{D/2})$
    $Hc(u,v) = circ(\frac{\sqrt{u^2+v^2}}{D/2\lambda l’})$
    截止频率$\rho_M = \frac{D}{2\lambda l’}$
    对于照相物镜成像系统
    像面截止频率$\rho_M = \frac{D}{2\lambda l’}$
    物面截止频率$\rho_M = \frac{D}{2\lambda l}$

非相干成像系统分析及光学传递函数

非相干系统的成像和光学传递函数（OTF）

  • 非相干系统的成像
    $I(x’,y’) = I_0(x’,y’)*h_I(x’,y’)$
    空间域中，像的强度分布是物的强度分布与点扩散函数的卷积
  • 光学传递函数（OTF）
    $H_I(u,v) = \frac{G_I(u,v)}{O_I(u,v)}$
    归一化
    $H(u,v) = \frac{G(u,v)}{O(u,v)}

OTF求法

  • 用OTF与CTF的自相关关系
    $ H_I(u,v) = F{|H_c(u,v)|^2}$
    $ H_I(u,v) = H_c(u,v)\otimes H_c(u,v)$
    $H(u,v) = \frac{\iint P^*(\xi,\eta)P(\xi+\lambda l’u,\eta+\lambda l’u)d\xi d\eta}{\iint |P(\xi,\eta)|^2d\xi d\eta}$

  • 图解
    分母：为光瞳的总面积
    分子：中心在原点与中心偏离 (-λl’u, -λl’v) 的两个光瞳的重叠面积
    $H(u,v) = \frac{两错开光瞳的重叠}{光瞳总面积}

  • 截止频率$\rho = 2\rho_0$

阿贝成像理论和波特实验

阿贝成像理论

  • 相干光照明的光学透镜成像，是由两次衍射形成的。光学像是干涉条纹的叠加

波特实验

  • 波特实验是阿贝成像理论的证明
  • 空间滤波器——放在频谱面上处理频谱的孔径或掩模
  • 空间滤波—— 对输入信息包含的各种空间频率成分施以振幅和相位调制,以实现特定的变换来改变像的结构。
  • 实验分析
    二维光栅
    (1)水平狭缝滤波—水平频谱代表物的垂直结构，像中只有垂直线条
    (2)垂直狭缝滤波—垂直频谱综合成像，像中只有水平线条
    一维光栅
    (1)只让0级谱通过：像面一片明亮无像
    (2)只让0级和±1级通过：像面在光栅几何像范围内呈现出一个周期与物光栅相同的余弦光栅
    (3)只让±2级通过：周期为物光栅一半
    (4)只挡0级：当a/d>0.5时，物体对比反转；a/d<0.5时，像形状与物相同，只是对比有些变化；a/d=0.5时，像面无光栅像

光学信息处理

  • 光学信息处理——用光学方法实现对输入信息的各种变换和处理
  • 光学方法实现变换或模拟运算：菲涅耳变换或傅里叶变换等；加、减、乘、除、微分、积分、卷积、相关等；编码，解码
  • 光学信息处理类型
    相干光学信息处理——相干光照明，复振幅分布信息
    非相干光学信息处理——非相干光照明，强度分布信息
    白光信息处理——白光照明
  • 光学处理特点（与数字图象处理比较）:并行处理,所有信息同时输入进行处理,适合图象的快速,实时处理;抗干扰能力强,以光速运算,操作方便,适合光学神经网络,光通信,光存储光互连,光计算等.
