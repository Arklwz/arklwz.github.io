应用光学——像差理论部分

发表于 2020-12-13 分类于 学习所悟 阅读次数：

  本章内容主要为涉及到像差的分析

  该部分《841工程光学基础》考纲内容如下

熟练掌握各种几何像差的概念和基本特征

球差

球差概念，轴向球差与垂轴球差，初级球差与高级球差

  • 球差概念
    (1)轴上点A发出的某孔径带的光线与近轴光线交于不同点，形成球差
    轴向球差$\delta L’$
    垂轴球差$\delta T’$
    (2)如对于某个孔径带有$\delta L’ = 0$时称为对这个孔径带消球差
    (3)存在球差时，在像面上会产生圆形弥散斑
  • 初级球差与高级球差
    $\delta L’ = A_1h_1^2 + A_2h_1^4 + A_3h_1^6… = a_1u_1^2 + a_2u_1^4 + a_3u_1^6…$
    $\delta T’ = B_1h_1^3 + B_2h_1^5 + B_3h_1^7… = b_1u_1^3 + b_2u_1^5 + b_3u_1^7…$
    其中第一项称为初级球差，后面各项依次称为二级球差、三级球差等。初级球差以外的各项统称为高级球差

球差曲线，具有初级球差和二级球差时的特征

  • 具有初级球差与二级球差时的特征
    (1)$h_1$或$u_1$很小很小时，为近轴区，$\delta L’ = 0$
    (2)$h_1$或$u_1$很小时，仅有初级量，称Seidel区，只需要计算一条边光即可确定公式中的系数
    (3)$h_1$或$u_1$有一定大小，四次项不可忽略,得仅有初级和二级像差时的公式
    $\delta L’ = A_1(\frac{h_1}{h_m})^2 + A_2(\frac{h_1}{h_m})^4$
    对边光校正球差，在0.707取得最大球差
    (4)$h_1$或$u_1$很大时,需要计算更多的光线
    例如到三级，这时当全孔径和0.707孔径校正球差后，0.85孔径带具有最大剩余球差
  • 球差曲线

单个折射球面的球差特征，三个无球差点、反常区与半反常区，齐明透镜设计

  • 单个折射球面的球差和球差分布
    会聚面产生负球差，发散面产生正球差，反常区和半反常区相反
    (1)球心到齐明点称反常区
    (2)顶点到球心称半反常区
  • 齐明点、齐明面与齐明透镜
    一对齐明点$L = \frac{n’ + n}{n}r, L’ = \frac{n’ + n}{n’}r$
    必为实物成虚像或虚物成实像。此时该面不产生球差，称齐明面。加同心面可得齐明透镜

    齐明透镜成组使用，常在测量仪器中用于扩大孔径

初级球差与孔径的关系，第一赛得和数，整体缩放对像差的影响

  • 初级球差，第一赛得和数
    仅有初级量的区域称赛得(Seidel)区，在此区域内有$sinU = u$
    $\delta L’ \propto u^2$
  • 整体缩放：指系统中所有线性量均乘以一个相同的倍数，而所有角度量不变
    根据比例关系，球差随整体缩放线性变化。这对于实际球差、高级球差和其他像差也正确

薄透镜与简单薄透镜系统的球差特征、最小球差形状

  • 正透镜恒产生负球差，负透镜恒产生正球差，当入、出射光线关于透镜对称时，球差取得极值（绝对值最小），此时的透镜形状为最小球差形状。
  • 单个薄透镜不可能消球差
    校正方法：双胶合透镜，微小间隔的双分离透镜

平行平板的球差

  • 近轴光线位移$\Delta l’ = (1 - \frac{1}{n})d$
  • 规律
    (1)平行平板恒产生正球差，只能以产生负球差的系统补偿之。当且仅当光线垂直于平板表面入射时球差零
    (2)平板厚则球差大
    (3)平板虽薄但孔径大，球差也大

轴外像差

正弦条件和等晕条件

  • 正弦条件
    $n’y’sinU’ = nysinU, \frac{h}{sinU’} = f’$
  • 等晕成像和等晕条件
    实际由于球差存在，只能要求近轴轴外点具有和轴上点A相同的成像缺陷。此时称等晕成像，需要满足的条件称等晕条件
    $1 - \frac{nsinU}{\beta n’sinU’} = \frac{\delta L’}{l_p’ - l’}$
    当球差为零时，等晕条件化为正弦条件。
    当不满足等晕条件时，轴上点与近轴轴外点成像缺陷不等，用正弦差表示

轴外像差

由于折射球面存在球差和像面弯曲，使轴外点衍生出一系列像差

  • 慧差：
    (1)当系统不满足等晕条件时，轴外点存在彗差。

    (2)上下光线的交点偏离主光线：子午彗差
    $K_t’ = \frac{1}{2}(y_a’ + y_b’) - y_p’$
    前后光线的交点偏离主光线：弧矢彗差
    $K_s’ = y_s’ - y_p’$
    (3)仅有彗差时，将形成彗星状的弥散斑
    (4)彗差和孔径、视场都有关, 且$K_{s0}, K_{t0} \propto u^2W$

  • 像散和像面弯曲:
    (1)子午细光束像点在主光线上，弧矢细光束像点在主光线和辅轴的交点上，两者之轴向距离为像散。子午细光束像点和弧矢细光束像点会偏离高斯像面。如果把各视场的子午细光束像点或弧矢细光束像点连起来，将会得到弯曲的像面，这就是像面弯曲。

    (2)像散为零时，子午细光束像点和弧矢细光束像点重合，但不与高斯像面重合，所以像面弯曲仍然存在，这种像面弯曲叫匹兹凡面弯曲
    (3)像散，场曲只与视场有关，且$x_t’, x_s’,\Delta x’ \propto W^2$

  • 畸变
    (1)阑位置移动，主光线与高斯像面交点高度变化，引起像的变形
    (2)畸变仅是像的变形，不影响像的清晰度
    (3)实际像高比理想像高大，称正畸变(枕形)，反之称负畸变(桶形)
    (4)相对畸变：$\frac{\delta y_p’}{y_0’} = \frac{y_p’ - y_0’}{y_0’}\times 100%$
    (5)特征：
    全对称系统（结构对称，物像对称），不产生畸变
    孔阑与之重合的接触薄系统，不产生畸变（主光线通过系统中心，沿理想方向射出）
    对于单薄透镜，光阑前移——负畸变，光阑后移——正畸变。因此，畸变与光阑位置有关
    (6)畸变只与视场有关，且$\delta y_p’ \propto W^3$

赛得和数

  • 初级球差-第一赛得和数
  • 初级慧差-第二赛得和数
  • 初级像散-第三赛得和数
  • 初级场曲-第三、四赛得和数
  • 初级畸变-第五赛得和数

匹兹凡面弯曲

  • 像散为0，任然存在场曲
  • 薄系统的匹兹凡和-第四赛得和数
  • 正负光焦度的分离是校正匹兹凡和的唯一方法

色差

位置色差

  • 由于同种材料对不同波长具有不同的折射率，对同一物方截距将算出不同的像方截距，导致位置色差
  • 对复色光成像的仪器要求对主色光校正单色像差，对成像光谱的两端校正色差

位置色差的度量与色差曲线

  • 近轴区$\delta l_{ch}’ = l_F’ - l_C’$
    一般有$\delta L_{ch}’ = L_F’ - L_C’$
    级数式：$\delta L_{ch}’ = a_0 + a_1h_1^2 + a_2h_1^4+… = \delta l_{ch}’ + a_1h_1^2 + a_2h_1^4+…$

位置色差与球差的比较

  • 位置色差和球差都是轴上点像差
  • 位置色差和球差都产生圆形弥散斑
  • 位置色差产生彩色圆形弥散斑，球差产生单色圆形弥散斑

倍率色差

  • 波长变化引起材料的折射率变化，继而引起光学系统的放大倍率变化，像的大小随之变化
  • $\delta y_{ch}’ = y_F’ - y_C’$

初级位置色差-第一色差和数

平行平板的初级位置色差

  • 所以平行平板恒产生正色差，当且仅当时不产生色差

单薄透镜与薄系统的位置色差

  • 可见正透镜恒产生负色差，负透镜恒产生正色差，色差的大小与物距(像距)有关

初级倍率色差-第二色差和数

单薄透镜与薄系统的初级倍率色差

  • $C_I$ 为0时，$C_{II}$自动校正为0
  • 当即孔阑与之重合时，不论位置色差如何都能校正倍率色差

色差的校正

  • 考虑双胶合及微小间隔的双分离组, 且满足
    $\begin{cases}
    \frac{\Phi_1}{v_1} + \frac{\Phi_2}{v_2} = 0\
    \Phi_1 + \Phi_2 = \Phi
    \end{cases}$
  • 有一定间隔d的分离透镜系统，满足
    $\begin{cases}
    \frac{\Phi_1}{v_1} + \frac{h_2^2}{h_1^2}\frac{\Phi_2}{v_2} = 0 \
    \Phi_1 + \frac{h_2}{h_1}\Phi_2 = \Phi
    \end{cases}$
    物方无穷远时：$\frac{h_2}{h_1} = 1 - d\Phi_1$

初级色差与孔径、视场的关系

  • 初级位置色差与孔径和视场无关
  • 初级倍率色差只与视场有关，且$\delta y_{ch}’ \propto W$
  • 近轴光存在初级位置色差
